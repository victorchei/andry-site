# Вимоги до сайту

## Наші послуги

- Сайт для пропозиції послуг вантажних перевезень та вантажникі

## Ключові моменти

- Лендінг (односторінковий сайт)
- Статичний сайт на стилях Bootstrap
- Мобільна та десктопна верстка у стилі Google теми
- SEO-індексація та оптимізація
- Всі необхідні атрибути для просування

## Додаткові вимоги

- Швидке завантаження сторінки
- Адаптивний дизайн для різних пристроїв
- Зручна навігація
- Контактна форма для замовлення послуг - потрібні посилання на соц мережі і номер телефону 
- Про компанію - коротка інформація про компанію, її досвід та переваги
